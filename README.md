## How to initialize/run the application

**Install the project dependencies:**
```
pip install -r requirements.txt
```

**To run the application:**
```
python oop_project.main.py
```

or

```
python -m oop_project.main
```

**To run the tests:**
```
pytest
```

or

```
pytest -v
```

or

```
pytest -vv
```
